# Recipe Selector Agent

An agentic workflow project for the hackathon.

## Overview

This project uses AI agents to power intelligent recipe selection and recommendations.

## Project Structure

- `AGENTS.md` - Agent definitions and configurations
- `docs/` - Project documentation and planning
- `src/` - Python source code and utilities
- `.instructions.md` - Custom Copilot instructions

## Getting Started

### Prerequisites
- Python 3.8+
- Git

### Setup

```bash
# Install dependencies
pip install -r requirements.txt

# Run the application
python src/main.py
```
