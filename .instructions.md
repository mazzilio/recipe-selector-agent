# Custom Copilot Instructions

You are assisting with the Recipe Selector Agent hackathon project.

## Context

This is an agentic workflow project where:
- Agents are defined in AGENTS.md
- Python utilities handle data processing and API calls
- Markdown files document agent specifications

## Guidelines

1. When modifying agents, update AGENTS.md with clear descriptions
2. For complex logic, implement in Python under `src/`
3. Keep documentation in `docs/` up to date
4. Use memory files (in `memory/`) to track team decisions and progress

## Key Files
- `AGENTS.md` - Agent definitions
- `src/main.py` - Main entry point
- `docs/` - Project documentation
