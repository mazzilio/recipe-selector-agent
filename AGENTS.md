# Agents

## Recipe Selector Agent

An AI agent that helps users discover recipes based on ingredients, dietary preferences, and cooking skill level.

### Capabilities
- Ingredient-based recipe recommendations
- Dietary restriction filtering
- Cooking complexity assessment
- Step-by-step cooking guidance

### Configuration

```
name: "Recipe Selector"
description: "Finds and recommends recipes"
model: "claude-3-5-sonnet"
```
