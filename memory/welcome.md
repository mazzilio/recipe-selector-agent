This is an example memory note. It's the kind of thing your agent writes when you tell it
to remember something.

Notice the shape: frontmatter at the top, plain language in the body, a `[[link]]` and a
`#tag` to connect it to the rest of the vault. Your agent's `capture-note` skill produces
exactly this format, and `recall` reads it back.

Try it: open OpenCode in this folder and say *"remember that our demo is on Friday"*, then
in a fresh session ask *"what do you know about the demo?"*. Watch a new file appear here.
